# 📊 Análisis de Factores de Riesgo de Extinción de Especies
Este repositorio contiene el análisis desarrollado para el proyecto **DataXperience**, enfocado en la modelación de especies mediante Ciencia de Datos.

## 📁 Contenidos del Proyecto
* **Código:** `analisis_dataxperience.R` (Script con limpieza de datos y regresión).
* **Documentación:** Documento explicativo detallado del proyecto.
* **Presentación:** Sustentación de los hallazgos.
